# api-box-wordpress
Skypostal api-box Wordpress plugin

## Installation instructions:
1. Go to your wordpress installation path and create a new directory under "wp-content/plugins" path with the directory containing the plugin source files.
-* Sample: wp-content/plugins/api-box-wordpress*
2. Enable the new plugin "Skypostal API-BOX" in the plugins section of your wordpress administration panel.
3. Configure the plugin access keys and URLS using the plugin's administration panel. Located in the main side bar as "Skypostal API-BOX".
4. View the list of available shortcodes and place them in any page or post previously created in your wordpress site.
